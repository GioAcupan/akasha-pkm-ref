Options I'm considering

1

Canvas path: don't embed the image at all

The .canvas file is the diagram — text nodes with tree structure, edges, positioning. Original photo sits in _assets/ for verification. No cropping needed.

2

Fallback path: embed the full page, describe the diagram

When the tiered classifier can't produce a Canvas, the note embeds ![[photo.jpg]] (full page) and the description tells you where to look. Acceptable since it's the fallback.

3

AI crop the diagram region

Vision LLM returns bounding box coordinates. Crop with ImageMagick before embedding. Full page stays in _assets/. Cropped version goes in the note. More complex pipeline but cleanest result.
