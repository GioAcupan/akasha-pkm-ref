Revised: One .md note always, .canvas embedded inside

The markdown note is always the entry point — canvas sits inside it

Pipeline (revised)
1. Classify image content
2. Generate outputs for this image:
├── Math
→ .md note (type: math) with LaTeX body
├── Diagram (structured)
→ .canvas file with transcribed nodes + edges
→ .md note (type: concept or existing type) with ![[diagram.canvas]] embed
├── Sketch / annotation
→ .md note (type: concept) with [[_assets/.../image.jpg]] link
└── Mixed (math + diagram)
→ .md note + .canvas, canvas embedded mid-body with ![[diagram.canvas]]
3. Store original → _assets/<session_id>/
4. Canvas files live alongside notes → Knowledge/<domain>/

Example: Mixed math + mind map page
