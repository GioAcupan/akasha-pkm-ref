Fallback chain: one attempt, then image-link

Should Mermaid failure fall back to Canvas attempt? Or skip straight to image?

Two options

A

Classify → try format → fail → image link

Simple chain. If the image is classified as a flowchart and Mermaid output is garbage, go straight to image-link. Don't re-try as Canvas — same image, same difficulty. Saves tokens and complexity.

B

Classify → try → fail → try other format → fail → image link

Retry chain. If Mermaid fails, try Canvas in case it was a misclassification. More robust against classifier errors, but burns tokens on retries and adds pipeline complexity.

My recommendation: go with A

The classifier prompt can tell the LLM to be conservative — "if unsure whether this is a mind map or a flowchart, prefer the format that matches the spatial layout." Misclassifications will be rare enough that the token cost of retries isn't worth it. Can always add retries later if we see a pattern of misclassifications.
